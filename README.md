# 57 club

лендинг музыкального лейбла - одностраничный сайт с темной минималистичной темой, плавными анимациями и интерактивными элементами

## о проекте

57 club — это лендинг музыкального лейбла, объединяющий артистов, услуги (биты, сведение, мастеринг), мерч и новости. сайт построен на чистых HTML, CSS и JavaScript без фреймворков и сторонних библиотек

## возможности

- **плавная навигация** — якорные ссылки со `scroll-behavior: smooth`
- **анимации появления** — блоки всплывают при прокрутке через `IntersectionObserver`
- **3D тилт карточек** — карточки мерча наклоняются за курсором
- **мозаичная сетка услуг** — плитки разного размера на CSS Grid
- **карточки артистов** — сетка с фото, жанром и ссылками на стриминговые сервисы
- **кнопка "наверх"** — появляется при прокрутке, плавно возвращает в начало
- **прогресс-бар чтения** — индикатор прокрутки сверху страницы
- **ховер-эффекты** — подъём карточек, свечение акцентом, инверсия ссылок
- **адаптив** — мобильная вёрстка через `@media`-запросы

## структура страницы

| section | descr |
|---|---|
| header | лого `57 club` по центру + навигация / burger на мобиле в будущем |
| news | лента новостей лейбла: релизы, события, анонсы |
| services | мозаичная сетка услуг — биты, сведение, мастеринг |
| merch | каталог мерча с ценами и кнопкой заказа |
| about label | манифест, текст про лейбл и сетка артистов |
| footer | внешние контакты и соцсети |

## технологии

- **HTML5** — семантическая разметка (`<article>`, `<time>`, `<nav>`)
- **CSS3** — flexbox, grid, переменные, анимации и переходы, `@media`
- **JavaScript (Vanilla)** — `IntersectionObserver`, обработчики событий, динамическая генерация элементов

## шрифты

- **Figtree** — основной текст
- **JetBrains Mono** — акценты, мета-информация, теги
- **Georgia** — заголовок лейбла

## запуск

специальной сборки не требуется, это статический сайт

1. склонируй или скачай репозиторий
2. открой `index.html` в браузере
