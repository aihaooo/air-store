# air store 

концептуальный лендинг интернет-магаза воздуха - учебный/коммерческий проект, одностраничный сайт с темной минималистичной темой, плавными анимациями и интерактивными элементами

## о проекте

air Store это лендинг для вымышленного магазина, продающего "воздух" как товар. сайт построен на чистых HTML, CSS и JavaScript без фреймворков и сторонних библиотек

## возможности

- **адаптивная темная тема** - угольный фон с лаймовым акцентом (`#C4F042`)
- **плавная навигация** — якорные ссылки со `scroll-behavior: smooth`
- **анимации появления** — блоки всплывают при прокрутке через `IntersectionObserver`
- **3D тилт карточек** — карточки каталога наклоняются за курсором
- **мозаичная сетка коллекций** — плитки разного размера на CSS Grid
- **кнопка "наверх"** — появляется при прокрутке, плавно возвращает в начало
- **ховер эффекты** — подъём карточек и свечение акцентом

## технологии

- **HTML5** — семантическая разметка
- **CSS3** — flexbox, grid, переменные, анимации и переходы
- **JavaScript (Vanilla)** — `IntersectionObserver`, обработчики событий, динамическая генерация элементов

## шрифты

- **Figtree** — основной текст
- **JetBrains Mono** — акценты и заголовки

подключены через Google Fonts

## запуск

2. `index.html` в браузере
